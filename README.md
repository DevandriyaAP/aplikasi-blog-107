# aplikasi-blog-107
Aplikasi blog berbasis Laravel untuk mengelola kategori, penulis, dan artikel. Dibuat dengan PHP, MySQL, dan Bootstrap. Langkah instalasi: kloning repositori, jalankan composer install, sesuaikan .env, migrasi, dan jalankan php artisan serve. Aplikasi bisa diakses di localhost:8000. Lihat dokumentasi lengkap di bawah ini.
